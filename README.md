# ASCII-Art
A C program to convert a PPM image to ASCII art in black and white.

:warning: The program only handles PPM P3.

Useful tools:
- [Convert Image to PPM P6](https://convertio.co/fr/)
- [Convert PPM P6 to PPM P3](https://thomasebsmith.github.io/ppm-converter/)

## Usage

```bash
gcc ascii_art.c -lm -o exec
./exec <PPM image> <options>
```

### Options

| Argument | Description                                                    |
| -------- | -------------------------------------------------------------- |
| -o       | Output file where the ASCII art is printed (default is stdout) |
| -mw      | ASCII art max width                                            |
| -mh      | ASCII art max height                                           |
| -bc      | Use a big ASCII char table (default)                           |
| -sc      | Use a small ASCII char table                                   |

## Examples

```bash
./exec ./examples/cats.ppm -o output
```
